# homelab
Self-hosted homelab powered by Proxmox, Docker and TrueNAS.

## Infrastructure

- Proxmox VE
- Ubuntu Server 24.04 LTS
- Docker & Docker Compose
- TrueNAS SCALE
- NFS storage

## Services

- Jellyfin
- WireGuard
- qBittorrent
- Prowlarr
- Radarr
- Bazarr

## Planned

- Sonarr
- Homepage
- Uptime Kuma
