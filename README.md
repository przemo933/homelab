# homelab
Self-hosted homelab powered by Proxmox, Docker and TrueNAS.

## Infrastructure

- Proxmox VE
- Ubuntu 24.04 LTS
- Docker & Docker Compose
- TrueNAS SCALE
- NFS storage

## Services

- Jellyfin
- WireGuard
- qBittorrent

## Planned

- Sonarr

- Homepage
